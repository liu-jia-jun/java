#网络编程

### ip
用于定位唯一一台主机

### 端口
用于定位一台主机上的某个进程,进程占用端口号，端口不可冲突

#### 端口的分类

1. 公有端口:0~1023
一些协议的默认端口
    + http:80
    + https:443
    + ftp:21
    + telent:23
2. 程序注册端口:1024~49151,用于分配用户或者程序
一些程序默认端口
    + Tomcat:8080
    + Mysql:3306
    + Oracle:1521

3. 动态，私有端口:49152~65535
